# AEGIS

Predict. Protect. Prevent.

AEGIS is an AI-powered safety platform that predicts risk before emergencies occur.

---

## Features

- AI Safety Score
- Smart Route Analysis
- Explainable AI
- Community Reports
- Verified Safe Spaces
- Smart SOS
- Voice Assistant
- Multilingual Support
- Guardian Dashboard
- City Safety Analytics
- Real-time Maps
